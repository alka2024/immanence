会員制eラーニングシステム

会員管理・動画配信・通知機能をすべて一体化した、教育機関向けのeラーニングプラットフォーム。 教育ビジネスにおけるオンライン学習の仕組みを、コストを抑えながら自社で運用できるよう設計しました。

開発の背景

既存のeラーニングサービスは月額コストが高い一方、自由なカスタマイズが難しく、また事業規模と比較するとサーバー容量が大きすぎてコスパが悪いという課題がありました。 そこで、自社内で動画配信・資料共有・進捗管理・リマインド通知などを統合したシステムを構築しました。

主な機能

  • 会員登録・ログイン（認証）機能
    • メールアドレス／パスワードによる安全な認証（Firebase Auth）
    • パスワードリセット・認証メール送信の自動化
    • 受講コースに応じたアクセス権限の自動付与
  • 動画・教材コンテンツ管理
    • YouTube API による限定公開動画の安全な配信
    • PDF・スライドなどの教材アップロードとダウンロード制御
    • コース／レッスン単位での教材整理と公開設定
  • 受講進捗の自動記録・可視化
    • 動画視聴完了率の自動計測（再生時間から判定）
    • レッスンごとの「未着手／学習中／完了」ステータス管理
    • weeklyレポートの自動生成・メール送信（GAS）
  • リマインド・通知システム（メール／LINE）
    • 一定期間学習が進んでいない受講者の自動検出
    • 個別状況に応じたパーソナライズ通知（未完了レッスンなど）
    • LINE Messaging API による開封率の高いプッシュ通知
    • 講師からの重要アナウンスの一斉配信
  • 管理者向けダッシュボード
    • コース別・会員別の学習進捗を一覧で可視化
    • 教材アップロード／公開設定のブラウザ操作
    • レポート送信の曜日・時間・テンプレート設定
  • 運用コスト最適化のための設計
    • Firebase Hosting による高速・低コスト運用
    • サーバー負荷に応じた自動スケール対応
    • 学習データの自社一元管理により改善PDCAが高速化
  • 拡張性の高いアーキテクチャ
    • コース追加・価格変更など運営側で柔軟に更新可能
    • クイズ／テスト／アンケートなどの追加機能に対応しやすい構造
    • Zoomや決済サービスなど外部APIとの連携にも対応可能

使用技術

  • Google Apps Script（GAS）
  • Firebase（Auth / Firestore / Hosting）
  • JavaScript
  • YouTube API・LINE Messaging API

導入効果

外部SaaSを使わずに会員制サービスを構築したことで、月額コストを大幅削減。 また、学習データを自社で一元管理できるため、改善PDCAのスピードが大幅に向上しました。
